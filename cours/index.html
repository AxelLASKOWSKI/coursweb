HTML

Qu'est-ce que le HTML?

HTML est le langage de balisage standard pour créer des pages Web.
HTML décrit la structure des pages Web en utilisant le balisage.
Les éléments HTML sont les éléments constitutifs des pages HTML.
Les éléments HTML sont représentés par des balises.
Les balises HTML étiquetent des éléments de contenu tels que "heading", "paragraph", "table", etc.
Les navigateurs n'affichent pas les balises HTML, mais les utilisent pour afficher le contenu de la page.
